to compile the game : $- make

the game has different versions. the last version is the version 3.

to execute the last version : $- ./projectA3

to execute other versions : $- ./projectB
			    $- ./projectA1
			    $- ./projectA2

note that the base version 0 (projectB) and the first version 1.0 (projectA1) does not have a graphical representation of the "wits"
